# Testing

## Testing responsibilities

Codex performs source and repository validation where possible.

The human owner performs:

- GameMaker compilation
- Runtime testing
- Visual inspection
- Input testing
- Gameplay-feel evaluation
- Export testing

## Required checks for every feature

- [ ] Project opens in GameMaker
- [ ] Project compiles
- [ ] No new runtime errors
- [ ] Main behavior works
- [ ] Edge cases tested
- [ ] Existing behavior still works
- [ ] No duplicate persistent controllers
- [ ] No missing resource references
- [ ] Git diff contains no unrelated changes
- [ ] Documentation updated when architecture changed
